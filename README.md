# AI Mock Interview

An AI-powered mock interview platform with frontend and backend components.

## Project Structure

- **frontend**: React-based user interface using Vite
- **backend**: Node.js server with Express for API handling

## Getting Started

### 1. Start the React Frontend
Open a terminal and run:
```bash
cd frontend
npm install
npm run dev
```

### 2. Start Convex Sync
Open a **second** terminal window and run:
```bash
cd frontend
npx convex dev
```
*(This connects your local frontend to the real-time cloud database).*

## Features

- User authentication
- CV upload and analysis
- AI-powered interview evaluation
- Interactive interview room
