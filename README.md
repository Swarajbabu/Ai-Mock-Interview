# AI Mock Interview

An AI-powered mock interview platform with frontend and backend components.

## Project Structure

- **frontend**: React-based user interface using Vite
- **backend**: Node.js server with Express for API handling

## Getting Started

### Frontend
```bash
cd frontend
npm install
npm run dev
```

### Backend
```bash
cd backend
npm install
npm start
```

## Features

- User authentication
- CV upload and analysis
- AI-powered interview evaluation
- Interactive interview room
